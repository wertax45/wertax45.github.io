技术实现与工具

1.技术栈

呈现方案：hexo + github pages + git + Async theme

该站点的技术栈清单，包括但不限于：

技术栈分类 具体技术 用途/功能 版本/备注
静态站点生成器 Hexo 将Markdown文档转换为静态HTML页面 Node.js驱动，支持主题和插件
前端框架 Node.js Hexo运行环境，JavaScript服务端运行时 建议使用LTS版本
模板引擎 EJS/Swig/其他 主题模板渲染，页面布局生成 Hexo默认支持EJS，主题可自定义
CSS预处理器 Stylus/Sass/Less 样式表编写和编译 Async主题可能使用Stylus
前端构建工具 Hexo内置构建系统 资源处理、页面生成、优化 基于Node.js流处理
主题框架 Async主题 提供博客UI界面和交互功能 基于Hexo主题规范开发
JavaScript库 jQuery/原生JS 前端交互和动态效果 Async主题可能包含
Markdown解析器 Marked/hexo-renderer-marked 将Markdown转换为HTML 可配置扩展语法
代码高亮 Hexo 自带的代码渲染 代码块的语法高亮显示
搜索功能 Local Search/Algolia 站内文章搜索 Async主题集成本地搜索
评论系统 Valine/Waline/Gitalk 文章评论功能 需要第三方服务支持
统计与分析 Google Analytics/Baidu Tongji 访问数据统计 通过主题配置添加
部署与托管 GitHub Pages 静态网站免费托管服务 自动SSL，自定义域名
版本控制 Git 源码和文章版本管理 分布式版本控制系统
持续集成/部署 GitHub Actions 自动化构建和部署流程 可配置workflow文件
SEO优化 sitemap.xml/robots.txt 搜索引擎优化 Hexo插件自动生成
数学公式 KaTeX 数学公式渲染 科技博客常用
流程图/图表 Mermaid/Flowchart.js 绘制流程图和图表 部分主题支持
图标字体 Font Awesome5/阿里图标库 界面图标展示 提升视觉体验

项目亮点

  1. 完全免费: GitHub Pages提供免费托管和SSL证书
  2. 快速部署: 推送代码到GitHub即可自动更新
  3. 版本可控: 所有文章和配置都有Git版本历史
  4. 高度可定制: 主题和插件系统提供无限扩展可能
  5. 性能优秀: 静态页面加载速度快

Q：图片太大，全放到github上加载慢怎么办？

A：直接使用互联网上的公开图片资源（如banner），或者在线图床，这里是两者都有。
